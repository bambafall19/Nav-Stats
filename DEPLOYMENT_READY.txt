🐛 DÉPANNAGE RAPIDE
═══════════════════════════════════════════════════════════════════════════

Le build échoue:
  → npm run build
  → rm -rf .next node_modules
  → npm install
  → npm run build

Les migrations Supabase échouent:
  → Vérifier que vous êtes connecté à Supabase
  → Vérifier que le projet est correct
  → Exécuter les migrations une par une

Le site ne se met pas à jour:
  → git log --oneline -5
  → Vérifier sur Vercel que le déploiement est en cours
  → Forcer un redéploiement sur Vercel

Les nouvelles fonctionnalités ne s'affichent pas:
  → Vider le cache du navigateur (Ctrl+Shift+Delete)
  → Vérifier que les migrations Supabase sont exécutées
  → Vérifier la console pour les erreurs
